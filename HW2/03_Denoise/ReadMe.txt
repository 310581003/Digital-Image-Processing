Operation : Denoise

File name : Denoise.c

Steps : 
	1. Open file "Denoise.c".
	2. Change the arguments in line 14 and 15. The first argumet of function "Denoise" is the path of input image, the second one is the path of output image, and the third one is to define the method you would like to use for de-noising (where mode=0 is Gaussian smoothing and mode=1 is Weighted-average filtering). As the followings, 
		Denoise ("../input3.bmp", "output3_1.bmp", mode=0);
		Denoise ("../input3.bmp", "output3_2.bmp", mode=1);
	3. Press compile and run.
