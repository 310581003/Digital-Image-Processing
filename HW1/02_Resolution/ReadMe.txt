Operation : Resolution

File name : bmp_resolution.c

Steps : 
	1. Open file "bmp_resolution.c".
	2. Change "the path for input file/output file" and "bits per pixel in one channel of output image"  in line 11. As the followings,

	3. Press compile and run.
