Operation : Scaling

File name : bmp_scaling.c

Steps : 
	1. Open file "bmp_scaling.c".
	2. Change "the path for input/output file" and "up_scaling_is_true"  in line 12. As the followings, 
		scaling("../input2.bmp", "output2_up.bmp", up_scaling_is_true=1); // first argument is the path to input file; second argument is the path to input file; third argument is up_scaling_is_true (upscaling:1 ; downscaling:0).
	3. Press compile and run.
