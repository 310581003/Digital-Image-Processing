Operation : Chromatic_Adaptation

File name : Chromatic_Adaptation.c

Steps : 
	1. Open file "Chromatic_Adaptation.c".
	2. Change the arguments in line 12, 13, 14 and 15. The first argumet of function "SoG" is the path of input image and the second one is the path of output image. As the followings, 
		SoG ("../input1.bmp", "output1_1.bmp");
    		SoG ("../input2.bmp", "output2_1.bmp");
    		SoG ("../input3.bmp", "output3_1.bmp");
    		SoG ("../input4.bmp", "output4_1.bmp");
	3. Press compile and run.
