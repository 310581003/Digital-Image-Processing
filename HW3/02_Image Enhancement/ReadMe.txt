Operation : Image Enhancement

File name : Image_enh.c

Steps : 
	1. Open file "Image_enh.c".
	2. Change the arguments in line 12, 13, 14 and 15. The first argumet of function "Retinex" is the path of input image and the second one is the path of output image. As the followings, 
		Retinex ("../1 Chromatic Adaptation/output1_1.bmp", "output1_2.bmp");
    		Retinex ("../1 Chromatic Adaptation/output2_1.bmp", "output2_2.bmp");
   		Retinex ("../1 Chromatic Adaptation/output3_1.bmp", "output3_2.bmp");
    		Retinex ("../1 Chromatic Adaptation/output4_1.bmp", "output4_2.bmp");
	3. Press compile and run.
